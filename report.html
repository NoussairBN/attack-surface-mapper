Recommandations de Correctifs (4)

CRITICAL Désactiver le mode debug

Ne jamais shipper avec debuggable=true. Gradle active automatiquement le debug en build debug.

android:debuggable="false"
HIGH Désactiver la sauvegarde ADB

Utilisez l'API BackupAgent pour contrôler précisément les données sauvegardées.

android:allowBackup="false"
CRITICAL Protéger les composants exportés

Définissez une permission custom avec protectionLevel='signature' sur chaque composant exporté.

android:permission="com.example.MY_PERMISSION"
MEDIUM Vérifier les deep links avec App Links

Utilisez Android App Links pour lier cryptographiquement votre app à votre domaine.

android:autoVerify="true" + assetlinks.json
